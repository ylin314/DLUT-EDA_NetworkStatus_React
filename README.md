# DLUT-EDA 校园网状态监控面板

## 快速访问

已部署到大连理工大学开发区校区校园网内网服务器，可通过以下地址访问：
- http://172.20.30.3/

## 安装和运行

```bash
# 安装依赖
npm install

# 启动
npm run dev

# 构建生产版本
npm run build

# 预览生产构建
npm run preview
```

## 功能特性

- 实时显示网络连接状态
- 显示账户信息（学工号、姓名）
- 显示 IP 地址和 MAC 地址
- 显示剩余流量和账户余额
- 智能识别终端设备类型
- 一键快捷操作：
  - 校园网登录
  - 自助服务系统访问
  - 网络连接注销
  - 状态信息刷新

## 使用说明

1. 连接到 DLUT-EDA 网络
2. 访问 http://172.20.30.3/
3. 界面将自动显示当前网络状态
4. 根据需要使用功能按钮：
   - 【跳转登录】：供设备无法自动弹出校园网登录页面时使用
   - 【自助服务】：手动进入用户自助服务系统查看余额
   - 【注销本机】：注销当前账号
   - 【刷新数据】：手动更新状态
